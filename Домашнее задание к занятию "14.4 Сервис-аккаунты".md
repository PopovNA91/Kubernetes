## Задача 1: Работа с сервис-аккаунтами через утилиту kubectl в установленном minikube

### Как создать сервис-аккаунт?

```
nikolay@nikolay-VirtualBox:~$ kubectl create serviceaccount netology
serviceaccount/netology created
```

### Как просмотреть список сервис-акаунтов?

```
nikolay@nikolay-VirtualBox:~$ kubectl get serviceaccounts
NAME       SECRETS   AGE
default    1         4d22h
netology   1         29s
nikolay@nikolay-VirtualBox:~$ kubectl get serviceaccount
NAME       SECRETS   AGE
default    1         4d22h
netology   1         32s
```

### Как получить информацию в формате YAML и/или JSON?

```
nikolay@nikolay-VirtualBox:~$ kubectl get serviceaccount netology -o yaml
apiVersion: v1
kind: ServiceAccount
metadata:
  creationTimestamp: "2022-05-15T12:11:49Z"
  name: netology
  namespace: default
  resourceVersion: "13948"
  uid: ac46446e-eee9-4c5f-8d62-28c19ca7f3f4
secrets:
- name: netology-token-tqqwf
nikolay@nikolay-VirtualBox:~$ kubectl get serviceaccount default -o json
{
    "apiVersion": "v1",
    "kind": "ServiceAccount",
    "metadata": {
        "creationTimestamp": "2022-05-10T13:52:22Z",
        "name": "default",
        "namespace": "default",
        "resourceVersion": "446",
        "uid": "33d17343-c018-4288-8adf-9796990754eb"
    },
    "secrets": [
        {
            "name": "default-token-kmrzf"
        }
    ]
}
```

### Как выгрузить сервис-акаунты и сохранить его в файл?

```
nikolay@nikolay-VirtualBox:~/serviceaccounts$ kubectl get serviceaccounts -o json > serviceaccounts.json
nikolay@nikolay-VirtualBox:~/serviceaccounts$ kubectl get serviceaccount netology -o yaml > netology.yml
nikolay@nikolay-VirtualBox:~/serviceaccounts$ ll
total 16
drwxrwxr-x  2 nikolay nikolay 4096 мая 15 15:13 ./
drwxr-x--- 44 nikolay nikolay 4096 мая 15 15:13 ../
-rw-rw-r--  1 nikolay nikolay  237 мая 15 15:13 netology.yml
-rw-rw-r--  1 nikolay nikolay 1145 мая 15 15:13 serviceaccounts.json
```

### Как удалить сервис-акаунт?

```
nikolay@nikolay-VirtualBox:~/serviceaccounts$ kubectl delete serviceaccount netology
serviceaccount "netology" deleted
```

### Как загрузить сервис-акаунт из файла?

```
nikolay@nikolay-VirtualBox:~/serviceaccounts$ kubectl apply -f netology.yml 
serviceaccount/netology created
nikolay@nikolay-VirtualBox:~/serviceaccounts$ kubectl get serviceaccount netology
NAME       SECRETS   AGE
netology   2         13s
```

## Задача 2 (*): Работа с сервис-акаунтами внутри модуля

Выбрать любимый образ контейнера, подключить сервис-акаунты и проверить
доступность API Kubernetes

```
nikolay@nikolay-VirtualBox:~/serviceaccounts$ kubectl run -i --tty fedora --image=fedora --restart=Never -- sh
If you don't see a command prompt, try pressing enter.
```

Просмотреть переменные среды

```
sh-5.1# env | grep KUBE
KUBERNETES_SERVICE_PORT_HTTPS=443
KUBERNETES_SERVICE_PORT=443
KUBERNETES_PORT_443_TCP=tcp://10.96.0.1:443
KUBERNETES_PORT_443_TCP_PROTO=tcp
KUBERNETES_PORT_443_TCP_ADDR=10.96.0.1
KUBERNETES_SERVICE_HOST=10.96.0.1
KUBERNETES_PORT=tcp://10.96.0.1:443
KUBERNETES_PORT_443_TCP_PORT=443
```

Получить значения переменных

```
K8S=https://$KUBERNETES_SERVICE_HOST:$KUBERNETES_SERVICE_PORT
SADIR=/var/run/secrets/kubernetes.io/serviceaccount
TOKEN=$(cat $SADIR/token)
CACERT=$SADIR/ca.crt
NAMESPACE=$(cat $SADIR/namespace)
```
